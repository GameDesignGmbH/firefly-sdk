regBitmapTexture()method 
protected function regBitmapTexture(id:String, path:String, autoScale:Boolean = true, keepStageAspectRatio:Boolean = false, vAlign:String, hAlign:String, checkPolicyFile:Boolean = false):void

Register bitmap based texture (PNG/JPEG) for loading.

Parameters

id:String — Unique identifier of the texture.
 
path:String — Path to the bitmap file.
 
autoScale:Boolean (default = true) — Specifies whether use autoscale algorithm. Based on design size and stage size texture will be proportionally scale to stage size. E.g. design size is 1024x768 and stage size is 800x600 the formula is var scale:Number = Math.min(1024/800, 768/600); Calculated scale is 1.28 and all bitmaps scale based on it.
 
keepStageAspectRatio:Boolean (default = false) — Specifies whether keep stage aspect ration. This property has effect to cropping of bitmap data.
 
vAlign:String — Vertical align type.
 
hAlign:String — Horizontal align type.
 
checkPolicyFile:Boolean (default = false) — Specifies whether the application should attempt to download a URL policy file from the loaded object's server before beginning to load the object itself.

Examples
The following code shows how to register different types of textures: 
     
public class GameTextureBundle extends TextureBundle
{
    override protected function regTextures():void
    {
        regFXGTexture(PauseButton);
        regBitmapTexture("bitmap_texture_1", "../textures/texture1.png");
        regBitmapTexture("bitmap_texture_2", "../textures/texture2.jpg");
        regATFTexture("atf_texture", "../textures/atf_texture.atf");
        regSWFTexture("swf_textures", "../textures/swf_textures.swf");
        regDragonBonesFactory("dragon_bones", "../textures/DragonBones.swf");    
    }
}
